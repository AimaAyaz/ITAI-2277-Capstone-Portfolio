# Phase 04 – Integrated System Prototype (Project Github Repository Link)

This phase combined all components into a working prototype.

## System Components
- YOLO person detection
- Thermal sensor temperature capture
- Heartbeat sensor integration
- Raspberry Pi hardware setup
- FastAPI model endpoint
- Node.js backend
- React touchscreen dashboard
- MongoDB for data storage

## Outcome
A fully functional pre-screening system capable of:
- Detecting individuals
- Measuring physiological signals
- Classifying illness risk
- Displaying results in real time

This phase represents the working prototype submitted for grading.
