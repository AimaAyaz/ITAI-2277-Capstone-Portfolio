## рџ“Њ **Project Overview**
Illness Detection is an AIвЂ‘powered preвЂ‘screening system designed to identify potentially sick individuals **before they enter shared environments** such as classrooms, offices, and public spaces.

The system combines:

- **Computer Vision (YOLO)** for person detection  
- **Thermal sensing** for temperature measurement  
- **HeartвЂ‘rate monitoring** via pulse sensor  
- **Logistic Regression** for illness risk classification  
- **Interactive dashboard** for realвЂ‘time screening  

This project was developed collaboratively over 15 weeks, following a structured multiвЂ‘phase workflow.

---

## рџЋЇ **Goal**
To build a **realвЂ‘time, contactless, multiвЂ‘sensor screening system** that helps detect abnormal physiological signals and reduce illness transmission in shared environments.

---

## рџ§  **How the System Works**
1. Detect person using YOLO  
2. Capture physiological data (temperature + heart rate)  
3. Send data to backend server  
4. Classify risk level using Logistic Regression  
5. Display results on a touchscreen dashboard
