## 📌 **Project Overview**
Illness Detection is an AI‑powered pre‑screening system designed to identify potentially sick individuals **before they enter shared environments** such as classrooms, offices, and public spaces.

The system combines:

- **Computer Vision (YOLO)** for person detection  
- **Thermal sensing** for temperature measurement  
- **Heart‑rate monitoring** via pulse sensor  
- **Logistic Regression** for illness risk classification  
- **Interactive dashboard** for real‑time screening  

This project was developed collaboratively over 15 weeks, following a structured multi‑phase workflow.

---

## 🎯 **Goal**
To build a **real‑time, contactless, multi‑sensor screening system** that helps detect abnormal physiological signals and reduce illness transmission in shared environments.

---

## 🧠 **How the System Works**
1. Detect person using YOLO  
2. Capture physiological data (temperature + heart rate)  
3. Send data to backend server  
4. Classify risk level using Logistic Regression  
5. Display results on a touchscreen dashboard
